Cloud Computing

Anfänge

Bereits in den 1960er Jahren gab es erste Ideen des Cloud Computing. Damals gab es anstatt Personal Computer noch sogenannte Mainframes, auf die von einem Terminal aus zugegriffen werden musste. Da diese Mainframes jedoch nur von einem Nutzer aufeinmal verwendet werden konnten, mussten viele weitere Benutzer lange Wartezeiten in Kauf nehmen. Ende der 1970er Jahren kam bereits die Idee auf über das ARPANET, den Vorreiter des heutigen Internet Computerressourcen zu Bündeln.

Cloud Computing im privaten Bereich

Erst innerhalb der frühen 2000er Jahren wurde das Cloud Computing im Privatenbereich interessant und von verschiedenen großen Konzernen aktiv erforscht. Pioniere sind hierbei zum Beispiel Amazon, Google oder IBM. Dadurch geriet das Cloud Computing erst richtig in Schwung. Durch spätere gemeinsame Projekte im Bereich des Cloud Computing konnte die Entwicklung solcher Cloud basierter System weiter beschleunigt werden.

Auswirkungen auf das Software-Engineering

Das moderne Cloud Computing ermöglicht das einfache Zusammenarbeiten von verschiedenen Computern aus. Zum Beispiel können mehrere Person simultan an derselben Datei arbeiten und die Auswirkungen des jeweils Anderen in Echtzeit sehen. Auch gegen das Verlieren von Dateien oder Änderungen hilft eine Cloud basierte Speichermethode.
